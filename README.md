# Список задач
Этот код создает простое графическое приложение на Python с использованием библиотеки **Tkinter**. Это приложение представляет собой список задач, который позволяет пользователю добавлять, удалять и перемещать задачи между двумя списками: *"Список задач"* и *"Список выполненных задач"*
